# 退休試算器

一個本機、純前端的退休試算小工具（台灣版）。台股／美股市值即時試算、收入支出與加薪、保底＋上漲退休目標、蒙地卡羅成功率、Coast FIRE。內建「教學」分頁解釋每個觀念。

## 隱私

**你的資料只存在你自己的瀏覽器（localStorage），不會上傳，沒有後端、沒有帳號。**
唯一會連網的是「抓收盤價」時把股票代號送去證交所／櫃買／Stooq 換價格（不含你持有幾股、多少錢）。想備份用右上角「匯出備份」。

## 怎麼跑（需要 Node 18 以上，建議 Node 20）

```bash
npm install
npm run dev
```

然後打開終端機顯示的網址（通常是 http://localhost:5173/）。

> 「抓取收盤價」需要用 `npm run dev` 跑才有效（開發伺服器會幫忙轉送、繞過瀏覽器的 CORS 限制）。
> 若改用 `npm run build` 打包成靜態檔，自動抓價會失效，但仍可手動輸入股價，其餘功能照常。

## 怎麼用

1. 「試算」分頁：填入持股（證券戶／代號／股數）、現金、年收入、支出、假設，即時看退休年齡與資產曲線。
2. 不確定觀念？先看「教學」分頁。
3. 想試「多花錢會不會延後退休」就拉支出滑桿；想看市場波動風險就按蒙地卡羅「執行模擬」。
